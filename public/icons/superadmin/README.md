# Superadmin Icons

This folder contains custom gradient icons specifically for the superadmin dashboard interface.

## Icon Usage

- `shield.png` - System Administration (security, system-wide settings) - **Custom gradient icon**
- `companies.png` - Companies Management (view and manage all companies) - **Custom gradient icon**
- `analytics.png` - Analytics & Reports (platform-wide metrics and reports) - **Custom gradient icon**
- `dashboard.png` - Regular Dashboard Access (return to standard user interface) - **Custom gradient icon**

## Custom Icon Mapping

When replacing with your custom icons:
1. `system_administration.png` → `shield.png`
2. `companies.png` → `companies.png` 
3. `analytics_and_reports.png` → `analytics.png`
4. `super_logo.png` → `dashboard.png`

## Notes

- All icons feature the teal-to-mauve gradient design matching the app theme
- Icons are displayed at 24x24px
- No filters applied - icons show in their original gradient colors
- Icons are used in `/app/superadmin/page.tsx`
- Background gradient containers complement the icon gradients
