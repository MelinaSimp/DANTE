# Domain Setup Checklist - driftai.studio

## ✅ What I See

Your domain `driftai.studio` is already added to Vercel! That's good.

---

## 🔍 What to Check

### 1. Is the Domain Assigned to Your Project?

**Check:**
- Click on `driftai.studio` in the list
- Or click the three dots (⋯) next to it
- See if it shows which project it's assigned to

**If not assigned:**
- Click "Add" or "Assign to Project"
- Select your "drift4" project

### 2. Check DNS Configuration

**Look for:**
- Does it show "Valid Configuration" or "Invalid Configuration"?
- Are there any warning messages?
- Does it show DNS records that need to be added?

**If DNS needs configuration:**
- Vercel will show you DNS records (A, CNAME, etc.)
- You need to add these at your domain registrar (where you bought driftai.studio)
- Common registrars: GoDaddy, Namecheap, Google Domains, etc.

### 3. Wait for DNS Propagation

**If DNS is configured:**
- It can take 10-30 minutes (sometimes up to 48 hours)
- The green icon suggests it might be working
- Try visiting `https://driftai.studio` again

---

## 🚀 Quick Test

1. **Try visiting:** `https://driftai.studio`
   - If it works → You're done! ✅
   - If it still shows ERR_FAILED → Continue below

2. **Check the domain details:**
   - Click on `driftai.studio` in Vercel
   - Look for any error messages or warnings
   - Check if it shows "Valid Configuration"

---

## 📝 What You Might Need to Do

### If Domain Shows "Invalid Configuration":

1. **Get DNS Records from Vercel:**
   - Click on the domain
   - Vercel will show you DNS records to add

2. **Add DNS at Your Registrar:**
   - Log into where you bought driftai.studio
   - Go to DNS settings
   - Add the records Vercel shows you

3. **Wait 10-30 minutes**

### If Domain Shows "Valid Configuration" but Still Not Working:

- **Just wait** - DNS propagation can take time
- **Try in incognito mode** - Sometimes browser cache causes issues
- **Try a different browser** - To rule out browser issues
- **Check in 30 minutes** - DNS changes take time

---

## ✅ Most Likely: You're Good!

If you see:
- ✅ Green icon next to the domain
- ✅ Domain listed in Vercel
- ✅ No error messages

**Then you probably just need to wait 10-30 minutes for DNS to propagate.**

Try visiting `https://driftai.studio` in 15 minutes. If it still doesn't work, check the domain details in Vercel for any error messages.

## 🆘 If Still Not Working After 30 Minutes

1. **Click on `driftai.studio`** in Vercel
2. **Look for error messages** or warnings
3. **Check DNS configuration** - Make sure records are added at your registrar
4. **Contact Vercel support** - They can help troubleshoot

**TL;DR:** Your domain is added. If it shows a green icon and no errors, just wait 10-30 minutes and try again. If there are errors, follow the steps above.
